StarLine

первоначальная настройка систем 6-го и 7-го поколений

Чтобы сэкономить время при выдаче автомобиля, настроить систему StarLine можно самому. Что для этого нужно? - карта владельца и номера SIM-карт, установленных в блок сигнализации

Шаг 1. Пополняем баланс SIM-карты на 100 руб.

Шаг 2. Устанавливаем номер владельца сигнализации и задаём пароль для управления с гостевых телефонов.
Для этого:
- придумываем любое 4-х значное число
- звоним по номеру телефона, указанному на карте владельца
- при установке соединения мы слышим: «Задайте пароль для управления с гостевых телефонов»
- на цифровой клавиатуре телефона вводим то 4-х значное число, которое мы заранее придумали
- после воода числа слышим звуковой сигнал и сбрасываем звонок.

Шаг 3. Скачиваем приложение StarLine2

Шаг 4. Отрываем по пунктирным линиям карту владельца. Перед нами 4 строчки:
1. Лонин для входа в мобильное приложение
2. Пароль для входа в мобильное приложение
3. Сервисный код.
4. Экстренный код

Шаг 5. Заходим в приложение по логину и паролю, указанными в карте владельца в мобильное приложение.

Шаг 6. Необязательно, но желательно после входа в мобильное приложение в разделе КОНТАКТЫ добавить свою эл. почту и поменять пароль. В этом случае логин для входа в приложение из карты владельца поменяется на название Вашего почтового ящика, а также не возникнет проблем для восстановления доступа в случае, если Вы забыли пароль или потеряли карту владельца сигнализации
